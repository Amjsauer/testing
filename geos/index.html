How to add an asset

  1. Select the Vehicles and Assets or the Assets option from the main menu.
  2. Click the + Asset button.
  3. Select Add single asset.
  4. Enter the asset name in the Asset name field.
  5. Enter the serial number of the GO device in the Serial number field.

    NOTE: Although this serial number is not required at this stage, MyGeotab operation will be limited until you add the serial number to your asset.
  6. Select an Asset Type Group. The built-in asset types are Trailer, Vehicle, Container, and Equipment.
  7. Select Groups to add the new asset to. Groups are used to organize your assets, users, and zones.
  8. Click Save New Asset when finished.

Additional Notes

  • Quick Import: Quickly import assets by pasting them in from Excel or CSV files.
    • Format: <Serial Number>, <Asset name/description>, <VIN/PIN>
    • Separate assets with a line break.
    • Serial number and VIN/PIN are optional. If you provide a serial number but the asset name/description is blank, we will add the serial number as the description.
  • See How to install the GO device if you have not yet installed your device.
  • Refer to the MyGeotab Product Guide for more information about assets.

How to add users

  1. Select the Administration > Users or People > Users & Drivers option from the main menu.
  2. Click the Add button.
  3. Enter a unique username or email address in the User (email) field.
    NOTE: Without a valid email address, users cannot reset their own password or receive exception notifications and reports by email.
  4. Enter the user's First Name and Last Name in the corresponding fields.
  5. Enter a Designation, Employee Number or Comments as needed.
  6. Select the Groups you want to add the user to.
  7. Select the appropriate group for Data Access. Users can only view data from the selected group's branch.
  8. In the Force password change at next login field:
    • Select Yes if an email address was used. This sends a password reset email containing a link to set the password. The link expires after one hour.
    • Select No if a username was used, then enter and confirm a password for the user.
  9. Select Basic authentication in the Authentication type field.
  10. Choose the appropriate Security clearance for the user.
  11. If the user is a driver, refer to the MyGeotab: How do I make a user a driver in MyGeotab and Compliance: How to configure a driver to meet ELD compliance regulations articles.
  12. Click Save when finished.

Additional Notes

  • Users can only be added by those with the appropriate security clearance.
  • Refer to the Standard Security Permissions document for details on configuring security permissions.
  • If the exact security permission is not set, the user will not have access to certain user settings. For example, if a user is configured to be a Driver and the security permission is Default User, the user will not have access to the HOS Settings tab.
  • Cannot see the Add button? Refer to the Unable to add or edit a user in MyGeotab article.

How to add zones

  1. Select the Map or Productivity > Zones > Zones option from the main menu.
  2. Click the Add Zone button on the Map page or the Add button from the Zones page.
  3. To create a zone using an address, type the address in the search bar.
  4. Click on the map to create the perimeter of the area. Click on the first added point to close the perimeter.

    TIP: Switch to Satellite view to better see roads while creating the zone.
  5. Enter the zone name.
  6. Select the groups to which the box is available under Publish to groups.
  7. Determine if the area is Visible on the map.
  8. Add types to group the zone with other related zones.
  9. Choose whether to Show Stops in Area to represent vehicles stopped within that area on the map with a star icon.
  10. Select an End Date to know when to stop recording data in the zone.
    NOTE: After the end date, the zone will not be deleted and can be reactivated at any time.
  11. Click on the Map View tab to adjust the shape, color and transparency of the area as needed.
  12. Click Save when finished.

Additional Notes

  • Make sure you associate the zone with the correct type.
  • Need to add a new zone type? See How to create a new zone type.
  • Check out the MyGeotab Product Guide for more tips.

How to pull a Watchdog Report

The Device Status Report, also called the Watchdog Report, provides an overview of the health of all devices in your fleet. An active device in good condition must communicate with MyGeotab at least once every 24 hours.

  1. Select the Administration > Report Setup... > Report Views or Reports > Report Setup > Report Views option from the main menu.
  2. Enter Device Status Report or Watchdog Report in the search bar.

    NOTE: If the report does not appear, select Show all types of reports and try the search again.
  3. Click Preview or View report next to the report name to view this report online.
  4. Click Export next to the report name to download the Excel template for this report.
  5. Click on Watchdog Report to adjust the report settings (add it to your dashboard, configure emailed reports, etc.).

Additional Notes

The Device Status Report is also accessible from the Assets or Vehicles & Assets page by clicking the Report dropdown menu and selecting Device Status Report or Watchdog Report.

NOTE: The report is only visible if the Show report in dropdown list option is set to Yes in the Reports view settings of the report.

How to display the location of a vehicle on the map

From the Map:

  1. Enter and select the desired vehicle name or GO device serial number in the search bar: all selected vehicles appear on the map.
  2. Use the Group Filter to display only assets from selected groups.
  3. Click on the desired vehicle in the asset list to view additional details and access its location on the map. By default, the list displays assets in the following order:
    • Active currently on the move.
    • Assets currently discontinued.
    • Assets arrested within an area.
    • Devices with which communication is broken.

From the Vehicles and Assets or Assets page:

  1. Enter the desired vehicle name, VIN or GO device serial number in the search bar.
  2. Select the desired vehicle to open the Asset Edit page.
  3. Click the View on map button.

How to enable built-in rules

Exception rules are conditions that describe the ideal behavior of a fleet. When a vehicle violates a rule, an exception is recorded in MyGeotab.

  1. Select the Rules & Groups > Rules or the Groups & Rules > Rules option from the main menu.
  2. Enter the name of the desired rule in the search bar.
  3. Select the desired rule for activation.
  4. Toggle the rule to On.
  5. Change sliders and values based on fleet needs and preferences.
  6. Click the Save button next to the ruler slider, if necessary.

Additional Notes

  • Want to learn more about vehicle classifications? Refer to the Truck Classifications page of Wikipedia.
  • Check out the MyGeotab Product Guide for more information about rules.

How to create a custom rule from a built-in rule

  1. Select the Rules & Groups > Rules or the Groups & Rules > Rules option from the main menu.
  2. Enter the name of the desired rule in the search bar.
  3. Click the pencil icon on the right.
  4. Enter a Name for the rule.
  5. Select which groups the rule applies to under Publish to groups.
  6. Click the Conditions tab.
  7. Click the Advanced Condition Editor icon to edit the values.
  8. Edit the rule conditions as needed.
  9. Click the Notifications tab.
  10. Add notifications to send when the rule is broken.
  11. Click Save when finished.

Additional Notes

  • Reprocess the data to see past exceptions. Refer to the How to reprocess data in MyGeotab article.
  • Consider disabling the built-in rule if you add new conditions to the custom rule.

How to create a Maintenance Reminder

  1. Select the Engine & Maintenance > Maintenance… > Recalls or the Maintenance > Reminders option from the main menu.
  2. Click the Add Reminder button.
  3. Enter the Reminder Name.
  4. Select the Maintenance Type for the reminder.
  5. Check the relevant boxes in the Repeats by: criteria.
  6. Set each reminder frequency according to your needs in Reminder date.
  7. Select the groups that can view or edit this reminder under Reminder Access.
  8. Click the Add Assets tab.
  9. Select the assets to attach the reminder to.
  10. For each asset:
    1. Select the Last Maintenance Date.
    2. Enter the Odometer reading at that time.
    3. Enter the Engine hours reading at that time.
    4. Click the Save button in the Actions column.
    NOTE: You cannot change the vehicle's service history once it is established, so make sure you have your latest service records on hand.
  11. Click the Notifications tab.
  12. Select the Type of notification generated by the reminder.
  13. Select the corresponding recipients for the reminder, if necessary.
  14. Set the conditions to trigger the reminder notification.
  15. Click the Add button.
  16. Click Save when finished.

Additional Notes

Refer to the MyGeotab Product Guide for more information on vehicle maintenance.

How to set up a Dashboard report

  1. Select the Administration > Report Configuration... > Report Views or the Reports > Report Setup > Report Views option from the main menu.
  2. Enter the name of the desired report in the search bar.
  3. Click the report name.
  4. Click the Dashboard tab.
  5. Click the Create New Chart button, if necessary.
  6. Select the type of chart to add to the dashboard.
  7. Enter the Name and a Description for the chart, if necessary.
  8. Set report display options:
    1. Select the groups that can see the report in their dashboard under Group dashboard viewers.
    2. Select who can see the report in their dashboard under Individual dashboard viewers.
    3. Select Yes for the Include me as the person who can see the dashboard option to add the report to your dashboard.
  9. Establish the report data configuration:
    1. Select the Report Data Refresh Period.
    2. Select the Next run date to refresh the report data.
    3. Under Belongs to, select the groups whose data you want to include in the report.
  10. Depending on the report, there may be other reporting options to set, such as special exceptions, zone types, or other thresholds.
  11. Click Save when finished.

    • El número de serie y VIN/PIN son opcionales. Si proporciona un número de serie pero el nombre/descripción del activo está en blanco, agregaremos el número de serie como descripción.

ES How to add a user

  1. Seleccione la opción Administración > Usuarios o Personas > Usuarios y controladores en el menú principal.
  2. Haga clic en el botón Agregar.
  3. Ingrese un nombre de usuario o dirección de correo electrónico únicos en el campo Usuario (correo electrónico).
    NOTA: Sin una dirección de correo electrónico válida, los usuarios no pueden restablecer su propia contraseña ni recibir notificaciones e informes de excepciones por correo electrónico.
  4. Ingrese el Nombre y Apellido del usuario en los campos correspondientes.
  5. Ingrese una designación, número de empleado o comentarios según sea necesario.
  6. Seleccione los grupos a los que desea agregar el usuario.
  7. Seleccione el grupo apropiado para Acceso a datos. Los usuarios solo pueden ver datos de la rama de grupos seleccionados.
  8. En el campo Forzar cambio de contraseña en el siguiente inicio de sesión:
    • Seleccione Sí si se utilizó una dirección de correo electrónico. Esto envía un correo electrónico para restablecer la contraseña que contiene un enlace para establecer la contraseña. El enlace caduca después de una hora.
    • Seleccione No si se utilizó un nombre de usuario, luego ingrese y confirme una contraseña para el usuario.
  9. Seleccione Autenticación básica en el campo Tipo de autenticación.
  10. Elija la autorización de seguridad adecuada para el usuario.
  11. Si el usuario es un conductor, continúe con Cómo convertir al usuario en conductor y Cómo configurar un controlador para cumplir con las regulaciones de cumplimiento de ELD.
  12. Haga clic en Guardar cuando haya terminado.

Notas Adicionales

  • Solo aquellos que tengan la autorización de seguridad adecuada pueden agregar usuarios.
  • Consulte el documento Permisos de seguridad estándar para obtener detalles sobre cómo configurar los permisos de seguridad.
  • Si no se establece el permiso de seguridad exacto, el usuario no tendrá acceso a determinadas configuraciones de usuario. Por ejemplo, si un usuario está configurado para ser Conductor y el permiso de seguridad es Usuario predeterminado, el usuario no tendrá acceso a la pestaña Configuración de HOS.
  • ¿No puedes ver el botón Agregar? Consulte No se puede agregar o editar un usuario en MyGeotab.

ES How to run a Watchdog Report

El Informe de estado del dispositivo, también llamado Informe de vigilancia, proporciona una descripción general del estado de todos los dispositivos de su flota. Un dispositivo activo y en buen estado debe comunicarse con MyGeotab al menos una vez cada 24 horas.

  1. Seleccione la opción Administración > Configuración de informes... > Vistas de informes o Informes > Configuración de informes > Vistas de informes en el menú principal.
  2. Ingrese el Informe de estado del dispositivo o el Informe de vigilancia en la barra de búsqueda.

    NOTA: Si el informe no aparece, seleccione Mostrar todos los tipos de reporte e intente la búsqueda nuevamente.
  3. Haga clic en Vista previa o Ver informe junto al nombre del informe para verlo en línea.
  4. Haga clic en Exportar junto al nombre del informe para descargar la plantilla de Excel para este informe.
  5. Haga clic en Reporte de estado de los dispositivos para cambiar la configuración (agregar el informe a su panel, configurar informes enviados por correo electrónico, etc.).

Notas Adicionales

También se puede acceder al Informe de estado del dispositivo desde la página Activos o Vehículos y activos haciendo clic en el menú desplegable Informe y seleccionando Informe de estado del dispositivo o Informe de vigilancia .

NOTA: El informe solo es visible si la opción Mostrar informe en la lista desplegable está configurada en Sí en la configuración de vista Informes del informe.

  1. Sélectionnez l'option Administration > Utilisateurs ou Personnes > Utilisateurs et pilotes dans le menu principal.
  2. Cliquez sur le bouton Ajouter.
  3. Entrez un nom d'utilisateur ou une adresse de courriel uniques dans le champ Utilisateur (courriel).
    REMARQUE : Sans adresse de courriel valide, les utilisateurs ne peuvent pas réinitialiser leur propre mot de passe ni recevoir par courriel des notifications d'exception et des rapports.
  4. Entrez le Prénom et le Nom de famille de l'utilisateur dans les champs correspondants.

    • Sélectionnez Oui si une adresse de courriel a été utilisée. Cela envoie un courriel de réinitialisation du mot de passe contenant un lien pour définir le mot de passe. Le lien expire après une heure.
    • Sélectionnez Non si un nom d'utilisateur a été utilisé, puis entrez et confirmez un mot de passe pour l'utilisateur.

Remarques supplémentaires

  • Si l'autorisation de sécurité exacte n'est pas définie, l'utilisateur n'aura pas accès à certains paramètres d'utilisateur. Par exemple, si un utilisateur est configuré pour être un Chauffeur et si l'autorisation de sécurité est Utilisateur par défaut, l'utilisateur n'aura pas accès à l'onglet Paramètres HOS.
